Gam3r Store


Este é um projeto de e-commerce desenvolvido com tecnologias modernas e eficientes para criar uma plataforma robusta e escalável. O objetivo é fornecer uma interface rápida e intuitiva para usuários, enquanto o backend gerencia dados de forma segura e eficiente.
Tecnologias Utilizadas
React: Para a construção da interface do usuário, oferecendo uma experiência interativa e responsiva.
Next.js: Framework React utilizado para renderização do lado do servidor (SSR) e geração de páginas estáticas, otimizando o SEO e o desempenho.
NestJS: Framework de backend Node.js eficiente para construir uma API robusta e escalável com TypeScript.
Tailwind CSS: Biblioteca de CSS utility-first para estilização rápida e customizável.
Prisma: ORM para gerenciamento de banco de dados, simplificando as consultas e mantendo a consistência dos dados.
PostgreSQL: Banco de dados relacional para armazenamento seguro e eficiente das informações de produtos, usuários e pedidos.

Influencers4You

Influence4You é uma plataforma de marketing de influenciadores que conecta empresas e influenciadores para colaborações em campanhas.
Com o Influence4You, as empresas podem encontrar influenciadores relevantes para promover seus produtos e serviços, enquanto os influenciadores podem descobrir oportunidades de parceria.
Tecnologias Utilizadas
Java: Usamos Java para o backend e a lógica de negócios.
TypeScript: O frontend é construído com TypeScript e React.
CSS: Estilizamos a interface com CSS e componentes reutilizáveis.
Funcionalidades Principais
Pesquisa de Influenciadores:
Os usuários podem pesquisar influenciadores com base em nichos, seguidores, localização e outras métricas.
Os influenciadores têm perfis detalhados com informações sobre suas áreas de interesse, público-alvo e taxas de colaboração.
Gestão de Projetos:
As empresas podem criar projetos e convidar influenciadores para colaborar.
Os influenciadores podem aceitar ou recusar convites e acompanhar o progresso dos projetos.
Monitoramento de Campanhas:
As empresas podem rastrear o desempenho das campanhas, incluindo métricas como alcance, engajamento e conversões.
Os influenciadores podem ver o desempenho de suas postagens e receber feedback das empresas.

Luzes

Descrição Geral: Este projeto demonstra um efeito visual onde luzes coloridas seguem o movimento do mouse, deixando um rastro suave e fluido, semelhante a uma lanterna. As luzes se movem dinamicamente e o rastro desaparece suavemente após um curto período. Demonstração: Você pode visualizar o efeito movendo o mouse pela tela no navegador. As luzes coloridas seguirão o cursor, criando um rastro com diferentes cores e efeitos de brilho. Tecnologias Utilizadas: HTML5: Estrutura básica do projeto. CSS3: Estilização das luzes, incluindo transições, sombras e efeitos de blur. JavaScript: Lógica para o movimento das luzes e criação dos rastros. Como Rodar o Projeto: Clone o repositório: git clone https://github.com/GeisilaCosta/luzes.git Navegue até o diretório do projeto: cd nome-do-projeto Abra o arquivo index.html no seu navegador preferido, arrastando o arquivo para uma aba do navegador ou usando um servidor local. Funcionalidades: Luzes coloridas seguem o movimento do mouse. Criação de rastros suaves que desaparecem gradualmente. Efeitos visuais de lanterna usando box-shadow e blur. Ajustes Possíveis: Cores das luzes: Alterar os valores background-color no CSS. Tamanho dos rastros: Ajustar width e height na classe .luz-rastro no CSS. Velocidade do rastro: Modificar a propriedade transition no CSS para mudar a duração das transições.

Barbearia

Descrição Geral: O Projeto FSW Barber é uma aplicação web desenvolvida com Next.js e React, destinada a ser uma plataforma para barbearias. A aplicação inclui uma página que lista as barbearias cadastradas com base em parâmetros de busca.
Componentes Principais:
Header: Renderiza o logotipo da aplicação e um botão de menu lateral que ativa o componente SidebarSheet.
Search: Permite ao usuário buscar barbearias por título ou serviço.
BarbershopItem: Renderiza informações e serviços de uma barbearia individual.
BarbershopsPage: Página principal que lista as barbearias cadastradas com base em parâmetros de busca.
Funcionalidades:
Listagem de barbearias com base em parâmetros de busca (título ou serviço). Componente de busca para encontrar barbearias específicas.
Cabeçalho com logotipo e menu lateral. Tecnologias Utilizadas:
Next.js: Framework para desenvolvimento de aplicações web com React. React: Biblioteca JavaScript para construção de interfaces de usuário. Prisma: ORM para interação com o banco de dados.
TypeScript: Linguagem de programação para desenvolvimento web.
Banco de Dados: Utiliza Prisma para interagir com o banco de dados, armazenando informações das barbearias, como nomes e serviços oferecidos.
Arquitetura: Baseada em componentes React, cada um responsável por uma funcionalidade específica. A página principal é o BarbershopsPage, com o Header gerenciando o cabeçalho e o Search facilitando a busca de barbearias.
